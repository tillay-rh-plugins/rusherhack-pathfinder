# rusherhack pathfinder

### pathfind an optimal path through 2b2t's highway system

---

This plugin scrapes https://www.desmos.com/calculator/kthhplyigl in order to get a current map of the highway system.

---

This rusherhack plugin requires the Baritone mod, as well as the Xaeros suite (Xaeros Minimap, Xaeros Worldmap, and XaerosPlus).

I tested this plugin with Baritone version 1.11.1-2, Xaeros Minimap version 25.3.12, Xaeros Worldmap version 1.40.16, and XaerosPlus version 2.31.5.
